# obhod (Универсальный)

Собственный обход DPI блокировок для Windows.
Работает для Discord, YouTube, Roblox, HTTPS и QUIC. 
Полностью автономный билд (все нужные файлы вшиты внутри).

### Установка (Сборка)
Требуется: Windows 10/11 x64, .NET 8 SDK (только для сборки).

1. Запустите `build.bat` для компиляции проекта.
2. Заберите готовый файл `obhod.exe` из папки `dist`.
3. Запустите от имени Администратора на любом ПК.
